# Нейросетевая модель машинного перевода с совместным обучением для русско-английско-казахской языковой группы

## Описание проекта
LangBridge — это веб-платформа класса LXP (Learning Experience Platform), направленная на повышение вовлеченности студентов в дистанционном обучении. 

Применение совместного обучения для русско-английско-казахской языковой группы (Trilingual Joint Training) — революционный подход, где казахский язык, имеющий ограниченные обучающие данные, получает знания от русского и английского языков через общее латентное пространство представлений. Это повышает качество перевода с участием казахского языка на 15-20% по сравнению с традиционными билингвальными моделями, делая приложение особенно эффективным для пользователей из Центральной Азии.

Ключевые особенности: 
* Многоязычный перевод: Поддержка 100+ языков (включая русский, английский, казахский)
* Множественные режимы ввода:
* Текстовый перевод — ввод с клавиатуры
Голосовой перевод — распознавание речи и озвучивание результата
Перевод с камеры (OCR) — сканирование текста с фото, вывесок, документов
Перевод документов — загрузка PDF, Word, Excel файлов.
* Персонализация: Тематические словари (медицина, бизнес, путешествия)
  
## Стек технологий
* Backend: Python 3.9+ + FastAPI PostgreSQL Redis (кэш переводов)
* Frontend: React + TypeScript Tailwind CSS Axios (API запросы
* AI: PyTorch 2.0+ Transformers (HuggingFace) Helsinki-NLP OPUS-MT или mBART-50 (готовые модели для ru-en-kk)
